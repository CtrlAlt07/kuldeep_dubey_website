# Homepage Design for Vedic Services

This repository contains the source code for the Vedic Services homepage. The site has been customized for **पं. कुलदीप दुबे** (नरसिंह घाट, उज्जैन) with a focus on phone call consultations.


## Running the Project
```bash
npm install   # install dependencies
npm run dev   # start development server (http://localhost:5173)
```

Feel free to customize further while keeping the design system intact.
